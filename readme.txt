== Description ==

**Content Generator** helps you automatically generate SEO-friendly blog posts and recipes using AI. Supports over 12 AI providers including OpenAI, Cohere, Anthropic, and more.

**Free features:**
- Supports multiple AI providers
- Blog & recipe post generation
- API key and model selection
- Custom post creation
- ACF field integration (optional)
- Easy-to-use admin UI

**Pro version adds:**
- Advanced tone/prompt controls
- Keyword optimization
- Image generation
- Scheduled posting
- Priority support

Learn more and upgrade: https://yemcoders.com/plugins/content-generator

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/content-generator/` directory, or install via the WordPress plugin screen.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Content Generator" in the WordPress admin menu.
4. Configure your AI provider API keys and generate posts.
